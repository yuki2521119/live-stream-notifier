# StreamLauncher

YouTubeチャンネルのライブ配信開始を検知し、自動でブラウザタブを開くChrome拡張。

---

## ローカルでのテスト方法

### 1. Chrome の拡張機能ページを開く

アドレスバーに以下を入力：

```
chrome://extensions/
```

または メニュー → 拡張機能 → 拡張機能を管理

### 2. デベロッパーモードを ON にする

右上の「デベロッパー モード」トグルをオンにする。

### 3. 拡張機能を読み込む

「パッケージ化されていない拡張機能を読み込む」をクリックし、このリポジトリのフォルダ（`manifest.json` があるフォルダ）を選択する。

### 4. 動作確認

- Chromeのツールバーに拡張機能のアイコンが表示されれば読み込み成功
- アイコンをクリックするとポップアップが開く

---

## コード変更後の更新方法

| 変更したファイル | 更新方法 |
|---|---|
| `background.js` | `chrome://extensions/` の 🔄 ボタンを押す |
| `popup.html` / `popup.js` | ポップアップを閉じて再度開くだけで反映 |
| `manifest.json` | 必ず 🔄 ボタンで再読み込み |

---

## デバッグ方法

### Service Worker のログを確認する

`chrome://extensions/` を開き、該当の拡張機能の「Service Worker」リンクをクリックするとDevToolsが開く。

- バックグラウンドの `console.log` の出力を確認できる
- `fetch` のエラーやアラームの動作を確認できる

### ポップアップのログを確認する

ポップアップを開いた状態で右クリック →「検証」を選択するとDevToolsが開く。
